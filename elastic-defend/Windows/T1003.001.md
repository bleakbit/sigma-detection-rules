##### Hunting in Security Onion
If the defender integration is present, an analyst can search for "process.Ext.api.name: OpenProcess OR process.Ext.api.name: MiniDumpWriteDump AND Target.process.executable:"C:\Windows\System32\lsass.exe" to see any time the MiniDumpWriteDump API was called with lsass as the target process.
```
process.Ext.api.name: OpenProcess OR process.Ext.api.name: MiniDumpWriteDump AND Target.process.executable:"C:\Windows\System32\lsass.exe"
```
An analyst can also search for different API names, including OpenProcess and ReadProcessMemory.
